YAFRA

The software technology source code framework covering Java, legacy C example using OSF/Motif, nodejs and javascript, mobile clients using cordova, angularjs and ionic, python flask, python user interface, php server, perl scripts, shell scripts. All coming with an continuous integration system build on shell scripts.

YAPKI

A public key infrastructure certification authority based on openssl and scripts. This Public Key Infrastructure (PKI) framework builds upon openssl.org and provides documentation and scripts to setup your own PKI. Read the how to document on yet another PKI (YAPKI). The YAFRA framework could be a business case for a dedicated application on YAPKI, but no efforts were done so far.

travelDB

A travel reservation system build on a relational database model. Provides a client/server approach build with C source code and using an OSF/Motif user interface. Python refactoring. Different scripts to maintain database. This is an example of a travel management application based on C code enhanced with gui elements and scripting. It does not relate as such to cayenne or other ORM technologies but shows how a legacy can be reused by ORM technologies. As well it shows how you can reuse C code with python.

About yafra.org

yafra.org provides a framework of software technologies as example. You can fork or clone bits and peaces and build your own application with adjustments as needed. The benefit for you is to have already a working example how to orchestrate different multi tier technologies. All sources are available under the Apache Open Source license. Only the public key infrastructure (PKI) example has some restriction due to the cryptography parts used. Read the GitHub wiki page for further information.
