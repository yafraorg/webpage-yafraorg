About yafra.org

yafra.org provides a framework of software technologies as example. You can fork or clone bits and peaces and build your own application with adjustments as needed. The benefit for you is to have already a working example how to orchestrate different multi tier technologies. All sources are available under the Apache Open Source license. Only the public key infrastructure (PKI) example has some restriction due to the cryptography parts used. Read the GitHub wiki page for further information.
